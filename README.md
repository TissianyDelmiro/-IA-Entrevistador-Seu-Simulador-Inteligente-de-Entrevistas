# -IA-Entrevistador-Seu-Simulador-Inteligente-de-Entrevistas
Este repositório é um Simulador Inteligente de Entrevistas, criado para ajudar na preparação para entrevistas de emprego na área de tecnologia. Ele simula perguntas técnicas, avalia as respostas e fornece um feedback personalizado, auxiliando no desenvolvimento contínuo de habilidades de comunicação e resolução de problemas.
