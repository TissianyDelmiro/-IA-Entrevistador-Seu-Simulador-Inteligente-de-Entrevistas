1. Pode me falar um pouco sobre o projeto que você desenvolveu nesse repositório?

Resposta:
O projeto é voltado para a análise de dados, onde utilizei Python e a biblioteca Pandas para manipular e limpar os dados. O objetivo foi trabalhar com um conjunto de dados real, realizando análises exploratórias e apresentando insights valiosos. Além disso, criei visualizações interativas utilizando Plotly e desenvolvi um aplicativo simples com Streamlit para exibir as análises de forma interativa, permitindo que o usuário explore os dados diretamente.

2. Quais tipos de dados você trabalhou no repositório e quais etapas do processo de análise de dados você seguiu?

Resposta:
No projeto, trabalhei com dados de vendas, que incluíam informações como preço, quantidade, e data das transações. As etapas seguidas foram:

Coleta de Dados: Inicialmente, fiz o download de um dataset público e o importei para análise usando Pandas.

Limpeza de Dados: Após isso, fiz o tratamento de valores ausentes e a remoção de duplicatas.

Análise Exploratória: Usei Pandas para explorar os dados e gerar estatísticas descritivas.

Visualização: Com Plotly, criei gráficos interativos para destacar padrões e tendências.

Interatividade: Com Streamlit, desenvolvi um aplicativo que permite aos usuários visualizar as análises em tempo real.

3. Qual foi o maior desafio enfrentado durante o desenvolvimento do projeto e como você o superou?

Resposta:
O maior desafio foi trabalhar com dados faltantes e inconsistentes. Durante a análise exploratória, percebi que alguns valores estavam ausentes ou mal formatados. Para resolver isso, utilizei o método fillna do Pandas para substituir os valores ausentes por uma média ou mediana, conforme apropriado, e corrigi os tipos de dados para garantir que as colunas estivessem no formato correto para a análise.

4. Você utilizou alguma técnica específica de visualização no seu repositório para ajudar na interpretação dos dados?

Resposta:
Sim, utilizei Plotly para criar gráficos interativos. Isso foi crucial para dar uma visão mais clara dos dados. Por exemplo, criei gráficos de barras para comparar as vendas por categoria e gráficos de linha para mostrar a evolução das vendas ao longo do tempo. A interatividade permitiu que o usuário visualizasse diferentes cortes dos dados facilmente, o que tornou a análise mais acessível e dinâmica.

5. Como você lida com o processo de apresentação de resultados de uma análise de dados?

Resposta:
Acredito que a apresentação dos resultados é tão importante quanto a análise em si. No projeto, utilizei Streamlit para criar um aplicativo onde os resultados das análises e visualizações pudessem ser apresentados de forma interativa. Isso não só facilitou a comunicação dos insights, mas também permitiu que o público explorasse os dados por conta própria, tornando a apresentação mais eficaz e envolvente.
